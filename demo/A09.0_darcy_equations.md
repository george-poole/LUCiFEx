# Darcy equations

## Mixed formulation

boundary value problem on $\textbf{x}\in\Omega$

$$\nabla\cdot\textbf{u} = 0$$
$$\textbf{u} = -\frac{\mathsf{K}(\textbf{x})}{\mu}\cdot(\nabla p - \textbf{f}\,)$$

essential boundary condition on $\textbf{x}\in\partial\Omega_{\text{E}}$

$$\textbf{n}\cdot\textbf{u} = u_{\text{E}}$$

natural boundary condition on $\textbf{x}\in\partial\Omega_{\text{N}}=\partial\Omega/\partial\Omega_{\text{E}}$

$$p = p_{\text{N}}$$

incompressibility constraint if $\partial\Omega_{\text{E}}=\partial\Omega\iff\partial\Omega_{\text{N}}=\varnothing$

$$\int_{\partial\Gamma}\text{d}\Omega~u_{\text{E}}=0$$

variational formulation $\forall(\textbf{v}, q)\in V_{\textbf{u}} \times V_p$

$$F(\textbf{u}, p;\textbf{v}, q) = \int_\Omega\text{d}\Omega~q(\nabla\cdot\textbf{u}) + \textbf{v}\cdot(\mu\mathsf{K}^{-1}\cdot\textbf{u}) - p(\nabla\cdot\textbf{v}) - \textbf{v}\cdot\textbf{f} + \int_{\partial\Omega_{\text{N}}}\text{d}\Gamma~p_{\text{N}}\,\textbf{v}\cdot\textbf{n}$$

## Streamfunction formulation

$$\textbf{u}=\nabla\times\boldsymbol{\psi} \iff \nabla\cdot\textbf{u}=0$$

$$\implies\nabla\times\left(\mu\mathsf{K}^{-1}\cdot\nabla\times\boldsymbol{\psi}\right) = \nabla\times\textbf{f}$$

$d=2$ Cartesian formulation

$$\boldsymbol{\psi}=\psi\textbf{e}_z\implies\textbf{u}=\frac{\partial\psi}{\partial y}\textbf{e}_x - \frac{\partial\psi}{\partial x}\textbf{e}_y$$

$$\textbf{f}=f_x\textbf{e}_x + f_y\textbf{e}_y$$

$$\nabla\cdot\left(\frac{\mu\mathsf{K}^{\mathsf{T}}\cdot\nabla\psi}{\text{det}(\mathsf{K})}\right)=-\frac{\partial(f_y)}{\partial x} + \frac{\partial(f_x)}{\partial y}$$
